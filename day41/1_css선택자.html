3. CSS 텍스트 관련 속성

강아지

한국어 '강아지'는 '개'에 어린 짐승을 뜻하는 '아지'가 붙은 말이다. 경상북도 사투리로는 강생이라고도 한다. 각 언어마다 강아지에 대한 별도의 명칭이 있는데, 영어로는 강아지를 'Puppy'(퍼피)라 부르며, 일본어로는 '小犬(코이누)'라 하며, 중국어로는 '小狗(샤오거우)' 혹은 '幼犬(유추안)'이라 한다.
